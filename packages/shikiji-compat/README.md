# shikiji-compat

Compatible build of `shikiji` to align with `shiki`.

[Changes and compatibility](https://github.com/antfu/shikiji#soft-breaking-changes)

## Install

```bash
npm i -D shikiji-compat
```

Or set the alias

```json
{
  "dependencies": {
    "shiki": "npm:shikiji-compat@0.6"
  }
}
```
