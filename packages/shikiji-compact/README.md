# shikiji-compact

Compactible build of `shikiji` to align with `shiki`.

> Work in progress. It's not 100% compatible yet, but feedback is welcome :)

## Install

```bash
npm i -D shikiji-compact
```

Or set the alias

```json
{
  "dependencies": {
    "shiki": "npm:shikiji-compact@0.6"
  }
}
```
